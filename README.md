# Hotel Management System
Hotel Management System is a website that allows the Hotel manager to handle all hotel activities online. Interactive GUI and the ability to manage various hotel room booking features conveniently.

## Features
*	Login authentication of users and admin.
*	Room's check-in through registration form.
* Keeping the record of all the customers.
* Availability/Non-Availability of rooms.
* Check-in and check-out record of customers
* Record of booked and unbooked rooms.
* Maintainance of rooms.
* Contact and feedback features.

## Technologies
* HTML
* CSS
* PHP
* MYSQL (Database)
* XAMPP server
* Sublime Text 3
